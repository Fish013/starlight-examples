Source map example

  1. Open your browser's console window and go to the Debug/Sources tab.
  2. Select the app.lua file from the list and you should see the Lua source code that is running.
  3. Add a breakpoint to line 3.
  4. Refresh the page.

Execution will stop at the breakpoint and you should be able to step through the source from there. In Firefox you can blackbox the Starlight runtime code so that you only see your Lua source being executed.
