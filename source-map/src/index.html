Source mapping with Starlight

  1. Open your browser's console window and go to the Debugger/Sources tab.
  2. Select the script/app.lua file from the list and you should see the Lua source code that is running.
  3. Add a breakpoint to line 3.
  4. Refresh the page.

Execution will stop at the breakpoint and you should be able to step through the source from there. You can blackbox the Starlight runtime code so that you only see the Lua source being executed.
